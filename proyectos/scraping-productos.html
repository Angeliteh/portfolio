Scraping de Productos

Aplicación web que extrae y compara precios de tarjetas RTX en tiempo real desde Amazon y MercadoLibre, permitiendo encontrar las mejores ofertas.

Descripción del Proyecto

Este proyecto es una aplicación web que utiliza técnicas de web scraping para extraer información de productos, específicamente tarjetas gráficas RTX, desde Amazon y MercadoLibre. La aplicación permite a los usuarios comparar precios, características y disponibilidad en tiempo real, ayudándoles a encontrar las mejores ofertas.

Características Principales

  • Extracción de datos en tiempo real de Amazon y MercadoLibre
  • Filtrado por modelo, precio y características
  • Comparación lado a lado de productos similares
  • Notificaciones de cambios de precio
  • Historial de precios para análisis de tendencias

Tecnologías Utilizadas

Python BeautifulSoup Selenium Flask MongoDB HTML/CSS JavaScript

Desafíos y Soluciones

Uno de los principales desafíos fue manejar los cambios frecuentes en la estructura de las páginas web objetivo. Para solucionar esto, implementé un sistema de selectores dinámicos que se adapta automáticamente a los cambios en el DOM. También utilicé Selenium para interactuar con páginas que cargan contenido dinámicamente mediante JavaScript.

Proceso de Desarrollo

El desarrollo de este proyecto siguió un enfoque iterativo, comenzando con un MVP que extraía datos básicos de una sola fuente. Gradualmente, fui añadiendo más funcionalidades y mejorando la precisión de los datos extraídos.

Arquitectura del Sistema

La aplicación está estructurada en tres componentes principales:

  1. Módulo de Scraping: Responsable de extraer datos de las fuentes web.
  2. API Backend: Procesa, almacena y sirve los datos extraídos.
  3. Interfaz de Usuario: Permite a los usuarios interactuar con los datos de manera intuitiva.

Resultados y Aprendizajes

Este proyecto me permitió profundizar mis conocimientos en web scraping, manejo de datos y desarrollo de APIs. Aprendí a superar desafíos como la detección de bots, los cambios en la estructura de las páginas y el procesamiento eficiente de grandes volúmenes de datos.
