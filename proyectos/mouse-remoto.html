Remote Mouse

Control remoto de PC desde dispositivos móviles con baja latencia

Descripción del Proyecto

Una aplicación móvil desarrollada en Flutter que permite controlar el mouse de un PC remotamente, con menor latencia y más características que las aplicaciones tradicionales. El servidor está implementado en Python.

Este proyecto surgió de la frustración con las aplicaciones existentes que presentaban alta latencia o funcionalidades limitadas. Mi objetivo fue crear una solución que ofreciera una experiencia fluida y responsiva, especialmente útil para presentaciones, control multimedia y uso general desde la comodidad del sofá.

Características Principales

  • Control preciso del cursor con baja latencia (menos de 50ms)
  • Gestos multitáctiles para funciones avanzadas
  • Teclado virtual con atajos personalizables
  • Modo presentación con controles específicos para diapositivas
  • Conexión segura mediante cifrado de extremo a extremo
  • Funciona en redes locales sin necesidad de internet

Tecnologías Utilizadas

Flutter Dart Python Socket.io PyAutoGUI WebSockets

Desafíos y Soluciones

Desafío: Reducir la latencia en la comunicación

La principal queja de los usuarios con aplicaciones similares era la alta latencia que hacía frustrante el uso.

Solución:

Implementé un protocolo de comunicación optimizado basado en WebSockets que reduce al mínimo el tamaño de los paquetes y utiliza un sistema de predicción de movimiento para compensar los retrasos en la red.

Desafío: Compatibilidad entre diferentes sistemas operativos

Lograr que la aplicación funcionara de manera consistente en Windows, macOS y Linux presentaba desafíos debido a las diferencias en cómo cada sistema maneja los eventos del mouse.

Solución:

Desarrollé una capa de abstracción en Python que normaliza los comandos del mouse y teclado, adaptándolos automáticamente al sistema operativo detectado, lo que permite una experiencia uniforme independientemente de la plataforma.

Resultados y Aprendizajes

Este proyecto me permitió profundizar en el desarrollo multiplataforma con Flutter y en la optimización de comunicaciones en tiempo real. La aplicación ha sido bien recibida por usuarios que necesitan controlar sus presentaciones o sistemas multimedia de forma remota.

Aprendí la importancia de la optimización a bajo nivel para aplicaciones que requieren respuesta en tiempo real, así como técnicas avanzadas de gestión de estado en Flutter para mantener una interfaz fluida incluso durante operaciones intensivas de red.
