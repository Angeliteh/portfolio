Descargador de Videos

Descarga videos de TikTok, Instagram y Facebook sin publicidad

Descripción del Proyecto

Una web y aplicación móvil para descargar videos de plataformas populares sin publicidad. Utiliza un servidor backend en Python para gestionar las peticiones y asegurar un servicio rápido y sin interrupciones.

Este proyecto nació de la necesidad de guardar contenido educativo y creativo de redes sociales para su visualización offline. A diferencia de otras soluciones, mi enfoque prioriza la privacidad del usuario, no muestra publicidad intrusiva y ofrece una experiencia fluida tanto en web como en dispositivos móviles.

Características Principales

  • Soporte para TikTok, Instagram y Facebook
  • Descarga de videos en alta calidad
  • Interfaz web responsive y aplicación móvil nativa
  • Procesamiento en servidor para mayor velocidad
  • Sin publicidad ni rastreo de usuarios
  • Opción para descargar solo audio de videos musicales

Tecnologías Utilizadas

React React Native Python FastAPI Redis Docker

Desafíos y Soluciones

Desafío: Cambios constantes en las APIs de redes sociales

Las plataformas de redes sociales cambian frecuentemente sus métodos de protección para evitar la descarga de contenido.

Solución:

Implementé un sistema modular que separa la lógica de extracción por plataforma, permitiendo actualizaciones rápidas cuando cambian los métodos de protección. También desarrollé un sistema de monitoreo automático que detecta fallos y alerta sobre la necesidad de actualizaciones.

Desafío: Gestión de alta demanda de tráfico

El servicio se volvió popular rápidamente, lo que generó picos de tráfico que sobrecargaban el servidor.

Solución:

Implementé un sistema de caché con Redis que almacena temporalmente los videos más solicitados, reduciendo la carga en el servidor. También desarrollé un sistema de colas para gestionar las solicitudes en momentos de alta demanda, asegurando que todas las peticiones sean procesadas sin sobrecargar los recursos.

Resultados y Aprendizajes

Este proyecto me permitió profundizar en el desarrollo full-stack, combinando una interfaz de usuario atractiva con un backend robusto y escalable. La aplicación ha alcanzado más de 10,000 usuarios activos mensuales y procesa miles de descargas diarias.

Aprendí la importancia de diseñar sistemas que puedan adaptarse rápidamente a cambios externos, así como técnicas de optimización para manejar grandes volúmenes de tráfico sin comprometer la experiencia del usuario.
