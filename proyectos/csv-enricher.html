El problema

La IA es poderosa. Usarla bien sobre hojas de cálculo es doloroso.

Profesionales de RH, ventas, finanzas y marketing trabajan cada semana con cientos o miles de filas de datos CSV no estructurados. El proceso manual de leer, juzgar y escribir resultados es lento, inconsistente e imposible de escalar.

Las soluciones existentes exigen habilidades de prompt engineering que la mayoría no tiene o son tan genéricas que no se adaptan a los criterios reales de cada empresa. La distancia entre “tengo estos datos” y “necesito aplicar inteligencia sobre ellos” era enorme.

La solución

Pipelines de IA conscientes del contexto de negocio, sin código.

CSV Enricher permite que el usuario defina su contexto de negocio una sola vez mediante MyProfile y lo reutilice en cientos de ejecuciones. La plataforma traduce esos formularios visuales a prompts optimizados para LLMs de forma invisible y consistente.

Un reclutador que necesita evaluar 500 CVs define el perfil ideal una sola vez, sube el CSV y recibe una lista priorizada con puntuaciones, razonamiento y recomendaciones en menos de 2 minutos.

Funciones clave
Plantillas de IA reutilizables

Plantillas por industria como evaluación de CVs, enriquecimiento de leads, procesamiento de facturas y clasificación de keywords, activables con un clic.

MyProfile, motor de inyección de contexto

Un formulario visual donde el usuario define sus criterios de negocio. Ese contexto se inyecta automáticamente en cada prompt para que la IA actúe como experta en ese dominio específico.

Pipeline de procesamiento por lotes

Los CSV grandes se dividen en lotes optimizados y se procesan sin saturar la API de OpenAI. Incluye rate limiting, reintentos y seguimiento de progreso.

Monetización completa con Stripe

Suscripciones, cobro por créditos, Stripe Checkout y manejo seguro de webhooks, todo integrado en producción.

Desafíos técnicos y soluciones

Inyección de contexto a escala

Problema

Los wrappers estándar de LLM generan resultados genéricos porque carecen de lógica de negocio específica. Los usuarios no saben escribir prompts de sistema complejos.

Solución

El motor MyProfile traduce un formulario visual con criterios de negocio a un meta-prompt estructurado y optimizado que se inyecta en cada lote de forma invisible.

Rate limiting y fallos de API

Problema

Enviar un CSV de 5,000 filas directamente a la API de OpenAI provoca límites de tokens, timeouts y fallos graves.

Solución

Una cola resiliente de procesamiento por lotes en FastAPI que fragmenta el CSV dinámicamente, monitorea headers de OpenAI, aplica backoff y reintenta chunks fallidos automáticamente.
