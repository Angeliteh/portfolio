Calculadora de Distancias

Matriz de distancias entre puntos utilizando Google Maps API

Descripción del Proyecto

Desarrollé una aplicación utilizando la API de Google Maps para calcular distancias relativas entre puntos, mostrando las distancias en una matriz. El proyecto involucra un backend robusto y una interfaz web intuitiva.

Esta herramienta surgió como solución para empresas de logística y transporte que necesitaban optimizar rutas entre múltiples ubicaciones. La aplicación permite a los usuarios ingresar hasta 25 ubicaciones diferentes y obtener una matriz completa de distancias y tiempos estimados de viaje entre todos los puntos.

Características Principales

  • Cálculo de distancias entre múltiples puntos (hasta 25)
  • Visualización en formato de matriz interactiva
  • Estimación de tiempos de viaje considerando tráfico en tiempo real
  • Exportación de resultados a Excel y PDF
  • Guardado de ubicaciones frecuentes
  • Optimización de rutas para reducir distancias totales

Tecnologías Utilizadas

JavaScript Node.js Express Google Maps API MongoDB Chart.js

Desafíos y Soluciones

Desafío: Limitaciones de la API de Google Maps

La API de Google Maps tiene límites en el número de solicitudes por día y en la cantidad de elementos que se pueden calcular en una sola solicitud.

Solución:

Implementé un sistema de caché inteligente que almacena los resultados de cálculos previos y los reutiliza cuando es posible. También desarrollé un algoritmo que divide las solicitudes grandes en bloques más pequeños para evitar superar los límites de la API, mientras mantiene la coherencia de los resultados.

Desafío: Visualización de grandes matrices de datos

Presentar matrices de 25x25 de manera clara y usable en diferentes dispositivos era un reto de diseño importante.

Solución:

Diseñé una interfaz de matriz interactiva con códigos de color para facilitar la interpretación de los datos. Implementé funcionalidades de zoom, filtrado y ordenación que permiten a los usuarios enfocarse en las partes relevantes de la matriz, incluso en dispositivos móviles.

Resultados y Aprendizajes

Este proyecto ha sido adoptado por varias empresas de logística, ayudándoles a reducir costos de transporte en un promedio del 15% gracias a la optimización de rutas. La aplicación procesa más de 1,000 cálculos de matrices diariamente.

Aprendí la importancia de optimizar el uso de APIs externas costosas, así como técnicas avanzadas de visualización de datos complejos. También profundicé en algoritmos de optimización de rutas como el problema del viajante (TSP) y sus aplicaciones prácticas en logística.
