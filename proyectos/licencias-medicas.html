Descripción General

Sistema desarrollado para una oficina del sector salud que automatiza el proceso de llenado de formatos de licencias médicas. Elimina la captura manual de datos repetitivos, reduce errores humanos y acelera significativamente el proceso de generación de documentos oficiales. El sistema procesa plantillas de Word con campos predefinidos y genera documentos completos listos para firma y entrega.

Características Principales

Múltiples Interfaces de Captura

Comandos, modo guiado, reconocimiento de voz y diccionarios predefinidos

Procesamiento Inteligente de Plantillas

Detección automática de campos con preservación del formato original

Gestión de Datos Médicos Especializados

Campos específicos para diagnósticos, períodos y información administrativa

Demostración en Video

Desafíos Técnicos y Soluciones

Detección de Campos Complejos

Problem

Los campos pueden estar en párrafos o tablas con formato complejo.

Solution

Búsqueda recursiva en todos los elementos del documento (párrafos, tablas, celdas).

Interfaz para Personal No Técnico

Problem

El personal médico necesitaba una interfaz simple e intuitiva.

Solution

Múltiples interfaces (guiada, por voz, comandos) adaptadas a diferentes niveles.

Preservación del Formato

Problem

Los documentos generados deben mantener exactamente el mismo formato que los oficiales.

Solution

Uso avanzado de python-docx para modificar solo el texto de los "Runs" específicos.
